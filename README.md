# Parallax Website Clone

## Description
A responsive single-page parallax scrolling website clone of the W3Schools Parallax Template, themed around **Travel Destinations**. Features fixed navigation, parallax background effects, contact form with Google Maps iframe, and full mobile responsiveness.

## Features
- **Semantic HTML5**: Uses `<header>`, `<nav>`, `<section>`, `<figure>`, `<form>`, etc.
- **Parallax Scrolling**: 4 sections with `background-attachment: fixed` (scroll on mobile for compatibility).
- **Fixed/Sticky Navbar**: With FontAwesome icons, hover effects, and hamburger menu on mobile.
- **Google Fonts (Montserrat)** and **FontAwesome icons**.
- **Google Maps iframe** in Contact section.
- **Responsive Design**: Media queries for tablets (768px) and mobiles (480px).
- **5+ CSS Classes**: `navbar`, `caption`, `border`, `bgimg-*`, `submit`, etc.
- **Pseudo-classes**: `:hover` on links/buttons, `.active`.
